
# RxOkHttp简介
一个对OkHttp封装的简单易用型HTTP请求和文件下载管理框架。更多详情请查看[WIKI]

* 简化[OkHttp](https://github.com/square/okhttp)使用
* 支持GET,POST,PUT,DELETE,HEAD,PATCH谓词
* 支持Activity和Fragment生命周期结束后终止请求
* 支持返回bean对象
* 支持返回json String数据
* 支持https请求
* 支持https证书访问
* 支持文件上传
* 支持全局params
* 支持全局header
* 支持http cancel
* ……
